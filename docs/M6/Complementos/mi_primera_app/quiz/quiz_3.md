# Taller Intermedio de Flutter (Proyecto Completo)

## Objetivo
Desarrollar una aplicación más robusta aplicando:
- Manejo de estado estructurado
- Consumo de APIs
- Persistencia de datos local
- Navegación con rutas
- Arquitectura básica (separación por capas)

---

## Proyecto del Taller
**App: "Gestor de Tareas Avanzado + API"**

La aplicación permitirá:
- CRUD completo de tareas (crear, leer, actualizar, eliminar)
- Guardar tareas localmente
- Consumir una API de tareas (simulada o real)
- Manejar estados de carga y error
- Navegación estructurada

---

## Requisitos Previos
- Haber completado un taller básico de Flutter
- Conocimiento de:
  - Widgets
  - setState
  - Navegación básica

---

## Duración Estimada
6 a 10 horas
