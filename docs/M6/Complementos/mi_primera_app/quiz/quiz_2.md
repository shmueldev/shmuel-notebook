# Taller Completo Básico de Flutter (Proyecto Guiado)

## Objetivo
Construir una aplicación funcional desde cero mientras aprendes los fundamentos esenciales de Flutter: UI, estado, navegación y organización básica del código.

---

## Proyecto del Taller
**App: "Gestor de Tareas Simple"**

La aplicación permitirá:
- Ver una lista de tareas
- Agregar nuevas tareas
- Marcar tareas como completadas
- Navegar entre pantallas

---

## Requisitos Previos
- Flutter instalado (`flutter doctor` sin errores críticos)
- Editor: VS Code o Android Studio
- Emulador o dispositivo físico

---

## Duración Estimada
4 a 6 horas

---

## Parte 1: Crear el Proyecto

### Paso 1
```bash
flutter create tareas_app
cd tareas_app
flutter run
```
