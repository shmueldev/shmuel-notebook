# Flutter

## ¿Cuando y por que fue creado?
Fue creado en 2015 con el nombre de sky luego en 2017 lanzaron su primer versión, fluter fue creado ya que en el mundo del desarollo movil, antes las empresas necesitaban dos equipos de desarollo uno para android y otro para IOS, con flutter un solo equipo puede desarollar para ambas plataformas más rápido y con menos costos.

## Configurar entorno de trabajo
Descargar las siguientes extensiones:
1. **Flutter:** Esta es la principal. Al instalarla, automáticamente se instalará también la de Dart.
1. **Tip extra:** Busca "Awesome Flutter Snippets" para escribir código mucho más rápido.

## Dark el lenguaje de Fluter
Dark es el lenguaje de flutter y al igual que C# o Java este es un lenguaje fuertemente tipado.
