### Pandas
Es una bibloteca de código abierto para y trabajar con datos **_tabulares_** (filas y columnas). Es rápida, potente y nos permte limpiar, filtrar y transformar datos de forma sencilla.

### Los dos tipos de estructuras (objetos)
Solo necesitamos aprender estos dos:
1. **_Series:_** Es como una sola columna de una tabla o un vector
    * **_Estructura:_** Tiene dos partes principales, el **_indice_** (la etiqueta de la fila) y los **_datos_** (el valor).
    * **_Regla de Oro:_** Todos los datos en una Serie deben de ser del mismo tipo (todos número o todos texto).
```py
# Una serie de temperaturas
temperaturas = pd.Series([22, 25, 21, 23], index=['Lun', 'Mar', 'Mie', 'Jue'])

print(temperaturas['Lun']) # Resultado: 22
```
1. **_DataFrame (la dimensión 2):_** Es la estructura reina. Es una tabla bidimencional (filas y columnas).
    * **_Estructura:_** Es básicamente un diccionario, cada **_llave_** es el nombre de la columna y cada **_valor_** es una **_Serie_**.
    * **_Flexibilidad:_** A diferencia de las series, un DataFrame puede tener distintos tipos de datos (una columna de texto, otra de números enteros, otras de fechas, etc)
```py
# Un DataFrame que une varias Series (Nombre, Edad, Ciudad)
datos = {
    'Nombre': ['Ana', 'Luis', 'Santi'],
    'Edad': [25, 30, 22],
    'Socio': [True, False, True]
}

df = pd.DataFrame(datos)
```

### Selección con `iloc` y `loc`
* `iloc` (Integer Location) seleciona por posición como una lista normal.
* `loc` (Label Location) seleciona por el nombre de la etiqueta (encabezados o índices).
Ejemplos con .iloc (por posición):
```py
# [Fila, Columna] -> Recuerda que en Python empezamos en 0
print(df.iloc[0, 0])      # El primer dato de la primera celda
print(df.iloc[0:5, :])    # Las primeras 5 filas y todas las columnas
print(df.iloc[:, [0, 2]]) # Todas las filas, pero solo columnas 0 y 2
```
Ejemplos con .loc (por nombre): Imagina un DataFrame con columnas "Nombre", "Edad" y "Ciudad".
```py
# Seleccionar solo la columna 'Nombre' para todas las filas
print(df.loc[:, 'Nombre'])

# Seleccionar filas donde el índice se llame 'Cliente_A' y la columna 'Edad'
print(df.loc['Cliente_A', 'Edad']) 
```

### Limpieza: Cómo borrar datos vacíos o arreglar errores.
En el mundo real los datos vienen "sucios": faltan nombres, hay valores `NAN` (Not a Number) o nulos. Aquí estan nuestras herramientas mágicas de emergencia:
1. **_Dectertar Nulos:_** `df.isnull().sum()` Nos indica cuantos datos faltan en cada columna.
1. **_Borrar nulos:_** `df_limpio = df.dropna()` Elimina una fila que tenga almenos un dato faltante.
1. **_Rellenar Nulos:_** `df['Edad'] = df['Edad'].fillna(0)` Si falta la edad le pone de manera automatica el 0 (o el promedio)
```py
# Eliminación
# Elimina filas donde CUALQUIER columna sea nula
df_limpio = df.dropna()

# Elimina solo si falta un dato en una columna específica (ej.'Email')
df.dropna(subset=['Email'], inplace=True)

# Imputación
# Rellenar con la media o la mediana (muy común en análisis numérico)
df['Edad'] = df['Edad'].fillna(df['Edad'].median())

# Rellenar con un texto genérico
df['Ciudad'] = df['Ciudad'].fillna('Desconocido')
```
### Correción de Errores y Datos Atípicos
Muchas veces los datos estan, pero son incorrectos (ej. una edad de 300 años o un correo sin "@"). Por eso hacemos lo siguiente:
* `Filtros lógicos:` Podemos sobrescribir valores que no cumplen con la lógica de negocio.
```py
# Corregir edades imposibles (mayores a 100 años) usando la mediana
df.loc[df['Edad'] > 100, 'Edad'] = df['Edad'].median()
```
* `Validación de strings:` Es muy útil para limpiar correos, nombres o categorías mal escritas.
```py
# Mantener solo filas que tengan un formato de email válido
df = df[df['Email'].str.contains('@', na=False)]
```
* `Ejemplo Integradi:` Este bloque de código nos muestra como pasar de un "dataset" sucio a uno listo para usar:
```py
import pandas as pd
import numpy as np

data = {
    'Nombre': ['Ana', 'Luis', 'Pedro', None, 'Marta'],
    'Edad': [25, 30, np.nan, 45, 200], # 200 es un error
    'Email': ['ana@mail.com', 'luis@mail.com', 'pedro@mail.com', 'error', 'marta@mail.com']
}

df = pd.DataFrame(data)

# 1. Borrar filas donde el nombre sea nulo
df.dropna(subset=['Nombre'], inplace=True)

# 2. Rellenar edad faltante con la mediana (30 en este caso)
df['Edad'] = df['Edad'].fillna(df['Edad'].median())

# 3. Corregir el outlier de 200 años (cambiarlo por la mediana)
df.loc[df['Edad'] > 100, 'Edad'] = df['Edad'].median()

# 4. Eliminar el registro con email inválido
df = df[df['Email'].str.contains('@')]

print(df)
```
### Las Agregaciones
Las **_agregaciones_** en Pandas consiste en reducir un conjunto de registros, (muchas filas), en único valor resumen (suma, promedio, conteo). El motor de este concepto se llama **_Split-Apply-Combine:_** Dividir los datos en grupos, aplicamos una operación y combinamos los resultados.
1. **_.groupby()_** Nos permite agrupar los registros para despues hacer funciones con estos
```py
import pandas as pd

data = {
    'Tienda': ['Norte', 'Sur', 'Norte', 'Este', 'Sur', 'Norte'],
    'Ventas': [100, 200, 150, 300, 250, 120],
    'Empleados': [5, 8, 5, 10, 8, 5]
}
df = pd.DataFrame(data)

# Sumar las ventas por tienda
ventas_por_tienda = df.groupby('Tienda')['Ventas'].sum()
print(ventas_por_tienda)
```
2. **_Las funciones de agregaciones más comunes_** en Pandas tenemos funciones integradas que podemos utilizar después del agrupamiento:
* `.sum()` Suma total de los valores.
* `.mean()` Promedio aritmético.
* `.count()` Cuenta cuántos registros hay (incluimos nulos si no especificamos).
* `.size()` Cuenta el número de filas por grupo.
* `.min()`/`.max()` Valores mínimos y máximos.    
3. **_Agregación múltiple con `.agg()`_** Muchas veces vamos a querer ver el promedio y la suma al mismo tiempo. Para eso utilizamos el metodo `.agg()`
```py

# Obtener el promedio y la suma de ventas por tienda
resumen = df.groupby('Tienda')['Ventas'].agg(['mean', 'sum', 'count'])
print(resumen)
```
4. **_Agregación personalizada por columna_** Podemos aplicar diferentes funciones a diferentes columnas usando un **_diccionario_**
```py
# Sumar ventas pero promediar el número de empleados por tienda
resumen_detallado = df.groupby('Tienda').agg({
    'Ventas': 'sum',
    'Empleados': 'mean'
})
print(resumen_detallado)
```
5. **_Agrupar por múltiples columnas_** Es como crear subcategorías dentro de una categoria principal.Imaginemonos que tenemos una empresea que tienen varias sucursales y en cada una venden productos diferentes. Si agrupamos por **_sucursal_**, veremos el total de la tienda, si agrupamos por **_sucursal y producto_**, podremos ver cuánto vendió cada producto en una tienda especifica.
```py
# Supongamos que tenemos este inventario
import pandas as pd

data = {
    'Sucursal': ['Norte', 'Norte', 'Sur', 'Sur', 'Norte', 'Sur'],
    'Producto': ['Teclado', 'Mouse', 'Teclado', 'Teclado', 'Mouse', 'Mouse'],
    'Ventas': [100, 50, 120, 110, 55, 45]
}
df = pd.DataFrame(data)
```
Para agrupar por mas de una columna, pasamos una lista `['columna1', 'columna2']` dentro del `groupby`:
```py
# Queremos saber el total de ventas por Sucursal Y por Producto
ventas_detalle = df.groupby(['Sucursal', 'Producto'])['Ventas'].sum()

print(ventas_detalle)
```
el resultado va a hacer un indice multinivel
```
Sucursal  Producto
Norte     Mouse       105  (50 + 55)
          Teclado     100
Sur       Mouse        45
          Teclado     230  (120 + 110)
```
un truco visual, podemos si queremos que el resultado parezca una tabla de excel, no escalonado, usamos, `.reset_index()`:
```py
resumen_tabla = df.groupby(['Sucursal', 'Producto'])['Ventas'].sum().reset_index()
print(resumen_tabla)
```
### Los Ìndices
Es el DNI o la etiqueta de cada fila, es lo que aparece en la columna de la izquierda, generalmente en negrilla, sirve para encontrar, identificar una fila rápidamente.
Por defecto **_Pandas_** le asigna a cada fila un número empezando dede 0, pero el índice puede ser cualquier cosa: nombres, fechas o inclusivo varias columnas a la vez. Muchas veces es mejor que el índice sea un una columna con un texto único como "ID" o "correo".
```python
# Convertir la columna 'Nombre' en el índice
df.set_index('Nombre', inplace=True)
```
#### ¿Qué es un "Multi-Índice" (Índice multinivel)?
Esto ocurre cuando agrupamos por varias columnas. Pandas "pega" esas columnas a la izquierda para organizar los datos de manera jerárquica. Si agrupas en por **_Sucursal_** y **_Producto_**, el índice se vuelve doble:

#### Dato curioso
Cuando hacemos una operación como `grupoby()` el resultado deja las categorías como **_índices_**, no como columnas normales, Esto hace dificil usarlas después para graficar o exportar en excel.
La solución magica es utilizar `.reset_index()` esta función empuja índice a su posición natural y pone número como (0, 1, 2) otra vez.
```py
# De un índice complejo a una tabla normal
df_normal = df_agrupado.reset_index()
```

## Matplotlib (Generar Graficas)
Para que las gráficas no sean solo "dibujos con líneas", hay que entender que **_Matplobit_** funciona como un lienzo para pintar, no solo le pedimos que dibuje, sino que tenemos que informarle qué pincel usar y dónde poner las etiquetas.

### 1. La anatomía de la gráfica (Los objetos)
En Matplobit existen varios términos que veremos siempre:
* **Figure (La Ventana)** Es el cuadro completo, el lienzo en blanco. Puede contener una o varias gráficas dentro.
* **Axes (La Gráfica)** Es el área especifica donde están los datos (los ejes x - y, el título, las barras). *Una figura puede tener varios Axes*.

### 2. Los elementos que "explican" la gráfica
Sin estos, nadie entendera nuestros análisis:
* **Title:** ¿De qué trata el gráfico?(Ej: "Ventas 2023").  
* **X-Label/Y-Label:** ¿Qué significan los números abajo y a la izquierda? (Ej: "Meses" y "Dólares").
* **Legend (Leyenda):** Si usas colores (ej: azul para hombres, rosa para mujeres), la leyenda explica qué es cada color.
* **Ticks (Marcas):** Son las rayitas de los números en los ejes. A veces hay que rotarlos si los nombres son muy largos.

### 3. Tipos de gráficas y cuándo usarlas
Cada tipo de dato necesita un "traductor" diferente:

| Gráfica            | ¿Para qué sirve?                         | Cuándo usarla (Tipo de dato)          | Ejemplo con el Titanic                                  |
|--------------------|------------------------------------------|---------------------------------------|---------------------------------------------------------|
| **Barras (Bar)**   | Comparar cantidades entre grupos.        | Categóricos (Texto) vs Numéricos.     | ¿Cuántos pasajeros había por cada clase (1ª, 2ª, 3ª)?   |
| **Barras Apiladas**| Ver la composición interna de un grupo.  | 2 Categorías vs 1 Numérico.           | De cada clase, ¿cuántos eran hombres y cuántas mujeres? |
| **Histograma**     | Ver la "forma" (distribución) de datos.  | 1 Numérico continuo.                  | ¿En qué rangos de edad se concentraba la mayoría?       |
| **Dispersión**     | Ver si una variable "mueve" a la otra.   | 2 Numéricos.                          | ¿A mayor precio del ticket, mayor supervivencia?        |
| **Líneas (Line)**  | Mostrar tendencias o evolución.          | Numérico vs Tiempo/Orden.             | ¿Cómo varió el precio promedio según la fecha?          |
| **Caja (Boxplot)** | Ver medianas, rangos y valores locos.    | 1 Categórico vs 1 Numérico.           | ¿Cuál era el rango de edad real en cada clase?          |
| **Pastel (Pie)**   | Ver proporciones de un total.            | 1 Categórico (pocas categorías).      | ¿Qué porcentaje del total eran hombres vs mujeres?      |
| **Mapa de Calor**  | Ver fuerza de relación entre variables.  | Matriz numérica (Correlaciones).      | ¿Qué tanto influye la edad y el precio en sobrevivir?   |
| **Violín**         | Mezcla de Boxplot e Histograma.          | 1 Categórico vs 1 Numérico.           | Ver la densidad de edades por clase o género.           |

### 4. ¿Por qué usamos Pandas con Matplotlib?
Pandas es muy útil porque tiene una función llamada `.plot()` que sirve de puente. En lugar de configurar todo a mano en Manplobit (que es muy largo), Pandas lo hace por nosotros:
1. **Pandas organiza:** Agrupa los datos y cuenta los valores.
1. **Matplotlib dibuja:** Toma esos resultados y los plasma en la pantalla.

### 5. Ejemplo funcional
Tipos de graficas
```py

df_plot.plot(kind='hist')   # 📉 histograma

```

### Correlaciones
Es la forma en la que medimos que tan relacionadas estas dos variables:
Ejemplo simple
```
Si sube danceability y también sube popularity
👉 correlación positiva 📈
Si sube speechiness y baja popularity
👉 correlación negativa 📉
Si no hay relación
👉 correlación ≈ 0
```
por ejemplo, vamos a responder esta pregunta `¿Qué características influyen en la popularidad?`
```py
# select_dtypes filtra las columnas, para que solo sea numericas y luego lo guarda en la variable numericas
numericas = df_limpio.select_dtypes(include='number')

```
